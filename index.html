Terminal Commands

  • ls: For listing files
  • cd: For changing directory
  • mkdir: For making directory
  • touch: For creating files
  • pwd: For printing working directory
  • rm: For removing files
  • cp: For copying files
  • cat: For concatenating files
  • mv: For moving files

Some Command Lines for Git

  • git version
  • git init
  • git add .
  • git status
  • git commit -m "first commit"
  • git commit -m "first commit"
